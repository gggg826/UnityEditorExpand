## **主要功能**
 - 快速切换历史场景
 - 快速UI编辑（隐藏除UI以外的层）
 - 集成SVN（非命令行打开，而是调用SVN窗口）
 - 资源被引用查找（未完，目前只是把结果以Log形式输出）
